# dashboards/anyerlin-ravelo-dashboard/
Proyecto de la asignatura de Big Data. Consiste en el análisis de un conjunto de datos numéricos utilizando Python y herramientas de Inteligencia Artificial para desarrollar un Dashboard que muestre estadísticas, patrones y visualizaciones clave. El objetivo es perfeccionar nuestras habilidades de análisis y corregir errores de forma colaborativa.
